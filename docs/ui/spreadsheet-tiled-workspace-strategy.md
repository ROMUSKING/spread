# Spreadsheet Tiled Workspace Strategy

Use a spreadsheet-first, tile-composable workspace. The grid remains the anchor; tiles are governed lenses over the same command-safe ERP model. Phase 0 may prepare canonical field identity, selected-record detail/transpose panel, tile metadata, subscription manifest concepts, and command-status visibility. Full tiling, AI tile, DuckDB tile, pgvector tile, connector dashboard, and custom dashboard designer are post-MVP only.
