# Keyboard and Focus Model

Use spreadsheet conventions plus optional tiling navigation: Tab/Shift+Tab cell focus, Enter edit/commit, Ctrl/Cmd+Enter submit, configurable tile focus movement, Ctrl/Cmd+K command palette, Esc leave edit/close transient tile.
