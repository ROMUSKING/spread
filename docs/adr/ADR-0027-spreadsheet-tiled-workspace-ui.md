# ADR-0027: Spreadsheet-First Tiled Workspace UI

Adopt a spreadsheet-first tiled workspace as the post-MVP UI direction. MVP may implement only simple selected-record detail/transpose scaffolding. Every editable tile uses command handlers; derived-plane tiles require RetrievalRevalidator or deterministic authority query before display.
